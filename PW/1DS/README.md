# ⚽ Real Madrid Website

Um site dedicado ao **Real Madrid**, o maior clube do mundo, desenvolvido como projeto da disciplina **Programação Web 1 (PW1)** no curso de **Desenvolvimento de Sistemas** da **E-Tech**.  
O objetivo do projeto foi recriar um site moderno e responsivo inspirado no clube, utilizando **HTML**, **CSS** e um pouco de **JavaScript**.
